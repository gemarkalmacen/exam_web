# Exam Web App

A Flutter Web application to display geolocation of IP addresses, including search history and a map view.

## Features

- Login with email & password
- Display user's IP geolocation
- Search IP addresses and display their location
- Side-by-side Geo Info and Map
- History list with checkbox and multi-delete
- Logout functionality (add-ons)

---

## Prerequisites

1. **Flutter SDK** (Managed with FVM)
2. **FVM** (Flutter Version Manager)
4. **Backend Api** (must set on this local ip address: http://localhost:8000)
5. A web browser (Chrome, Edge, or Firefox)
